# Interview Scripts

Guided conversations to develop story elements through structured questioning.

## Usage
1. Choose an interview script
2. Run with:
```bash
python Core/Scripts/run_interview.py [script_name]
```
3. Answers are saved to appropriate intake folder

## Available Interviews
- `character_development.md`
- `character_voice.md`  
- `location_creation.md`
- `research_questions.md`
- `style_refinement.md`
